## Ballfield

SDL2 demo using the SDLAda binding.

## Build and Run

```sh
gprbuild
./ballfield PLATFORM=macosx
```

## Requirements

* Ada 2012 compiler
* GPRbuild
* [SDLAda](https://github.com/Lucretia/sdlada)
